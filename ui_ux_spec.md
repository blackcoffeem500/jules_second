# UI/UX Specification

## 1. 画面構成
### ゲーム画面 (Main Game Screen)
- **Top Bar:** 現在のスコア、レベル、残りライフ（ハートアイコン）を表示。
- **Play Area (Canvas):** - 上部：ブロック配置エリア（10x5 などのグリッド）。
    - 中央：ボールが移動する空間。
    - 下部：プレイヤーが操作するパドル。
- **Overlay:** ゲームオーバー時やクリア時に、「再試行」または「次のレベルへ」のボタンを表示。



## 2. インタラクション
- **パドル操作:** 画面のどこを触っても左右のスワイプでパドルが追従する（絶対座標ではなく、移動量に基づく相対移動が望ましい）。
- **ゲーム開始:** 初回タップでボールがパドルから射出される。
- **視覚効果:** - ブロック破壊時にパーティクル（火花）が出る。
    - 壁やパドルに当たった際に微かな画面の揺れ（Screen Shake）を発生させる。

## 3. カラーパレット
- 背景: ダークネイビー (#0F172A)
- パドル: ネオンブルー (#38BDF8)
- ボール: ホワイト (#FFFFFF)
- ブロック: レベルや耐久度に応じて色を変化（赤、オレンジ、黄、緑）
