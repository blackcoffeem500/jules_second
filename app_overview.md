# Project Overview: NeoBreakout Android

## 1. 目的
スマホ向けに、ブラウザ（Javascript）を使用したシンプルかつ中毒性のあるブロック崩しゲームを開発する。

## 2. ターゲット
- Androidスマートフォンユーザー
- 隙間時間にプレイできるカジュアルゲームを求める層

## 3. 主要機能 (MVP)
- **ゲームプレイ:** パドルを操作し、ボールを打ち返して全てのブロックを破壊する。
- **操作性:** 画面下部のスワイプによる直感的なパドル操作。
- **レベルデザイン:** 難易度が段階的に上がる複数のステージ構成。
- **状態管理:** スコア、残りライフ、ゲームのフェーズ（開始前、プレイ中、一時停止、ゲームオーバー）の管理。

## 4. 成功の定義
- 60FPSで滑らかに動作する描画。
- 物理演算（反射）に違和感がないこと。
- 画面サイズ（アスペクト比）に関わらず正しく描画されること。
